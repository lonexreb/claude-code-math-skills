# IMO 2023 Problem 1

## Problem

Determine all composite integers `n > 1` that satisfy the following property:

> If `d₁, d₂, …, dₖ` are all the positive divisors of `n` with `1 = d₁ < d₂ < … < dₖ = n`, then `dᵢ` divides `dᵢ₊₁ + dᵢ₊₂` for every `1 ≤ i ≤ k − 2`.

## Reference solution sketch

Answer: prime powers `n = pᵏ` with `k ≥ 2`.

**Direction 1 (prime powers work).** If `n = pᵏ`, the divisors are `1, p, p², …, pᵏ`. Then `pⁱ | pⁱ⁺¹ + pⁱ⁺² = pⁱ⁺¹(1 + p)`. Trivially yes.

**Direction 2 (only prime powers).** Suppose `n` has at least two distinct primes `p < q`. Show that the smallest two divisors `> 1` are `p` and `q` if `q < p²`, or `p` and `p²` if `p² < q`. Either way, derive a contradiction with the divisibility chain. The harder case is `q < p²`: then `d₁ = 1, d₂ = p, d₃ = q`, so `1 | p + q` is automatic — go further: `d₄`. This is where the case analysis bites.

## Notes

- This is a **classification** problem — the answer is a set, and you must prove both inclusions.
- **Critic:** must verify the case analysis is exhaustive. IMO problems often have a "missed case" that produces a tempting wrong answer.
- The problem rewards the **strategist** picking "structure of divisor lattice" as the framing rather than ad-hoc number crunching.
