Definition 0.3.21

**Definition 0.3.21.** A relation _R_ on a set _𝐴_ is said to be

- (i) _Reflexive_ if _𝑎 R 𝑎_ for all _𝑎_ ∈ _𝐴_ .

- (ii) _Symmetric_ if _𝑎 R 𝑏_ implies _𝑏 R 𝑎_ .

- (iii) _Transitive_ if _𝑎 R 𝑏_ and _𝑏 R 𝑐_ implies _𝑎 R 𝑐_ .

If _R_ is reflexive, symmetric, and transitive, then it is said to be an _equivalence relation_ .
