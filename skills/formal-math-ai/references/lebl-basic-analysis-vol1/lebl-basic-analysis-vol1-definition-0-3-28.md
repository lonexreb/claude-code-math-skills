Definition 0.3.28

**Definition 0.3.28.** We write

| _𝐴_ | ≤| _𝐵_ |

if there exists an injection from _𝐴_ to _𝐵_ . We write | _𝐴_ | = | _𝐵_ | if _𝐴_ and _𝐵_ have the same cardinality. We write | _𝐴_ | _<_ | _𝐵_ | if | _𝐴_ | ≤| _𝐵_ |, but _𝐴_ and _𝐵_ do not have the same cardinality.

We state without proof that _𝐴_ and _𝐵_ have the same cardinality if and only if | _𝐴_ | ≤| _𝐵_ | and | _𝐵_ | ≤| _𝐴_ |. This is the so-called Cantor–Bernstein–Schröder theorem. Furthermore, if _𝐴_ and _𝐵_ are any two sets, we can always write | _𝐴_ | ≤| _𝐵_ | or | _𝐵_ | ≤| _𝐴_ |. The issues surrounding this last statement are very subtle. As we do not require either of these two statements, we omit proofs.

The truly interesting cases of cardinality are infinite sets. We will distinguish two types of infinite cardinality.
