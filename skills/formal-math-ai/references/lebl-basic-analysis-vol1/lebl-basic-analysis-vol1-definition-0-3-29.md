Definition 0.3.29

**Definition 0.3.29.** If | _𝐴_ | = |ℕ|, then we say _𝐴_ is _countably infinite_ . If _𝐴_ is finite or countably infinite, then we say _𝐴_ is _countable_ . If _𝐴_ is not countable, then _𝐴_ is said to be _uncountable_ .

The cardinality of ℕ is usually denoted as ℵ0 (read as aleph-naught)
