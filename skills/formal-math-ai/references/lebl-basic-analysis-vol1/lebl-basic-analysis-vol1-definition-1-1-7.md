Definition 1.1.7

**Definition 1.1.7.** A field _𝐹_ is said to be an _ordered field_ if _𝐹_ is also an ordered set such that

   - (i) For _𝑥, 𝑦, 𝑧_ ∈ _𝐹_ , _𝑥 < 𝑦_ implies _𝑥_ + _𝑧 < 𝑦_ + _𝑧_ .

   - (ii) For _𝑥, 𝑦_ ∈ _𝐹_ , _𝑥 >_ 0 and _𝑦 >_ 0 implies _𝑥𝑦 >_ 0.

- If _𝑥 >_ 0, we say _𝑥_ is _positive_ . If _𝑥 <_ 0, we say _𝑥_ is _negative_ . We also say _𝑥_ is _nonnegative_ if _𝑥_ ≥ 0, and _𝑥_ is _nonpositive_ if _𝑥_ ≤ 0.
