Definition 4.2.1

**Definition 4.2.1.** Let _𝑆_ ⊂ ℝ be a set and let _𝑓_ : _𝑆_ → ℝ be a function. The function _𝑓_ is said to have a _relative maximum_ at _𝑐_ ∈ _𝑆_ if there exists a _𝛿>_ 0 such that for all _𝑥_ ∈ _𝑆_ where | _𝑥_ − _𝑐_ | _< 𝛿_ , we have _𝑓_ ( _𝑥_ ) ≤ _𝑓_ ( _𝑐_ ). The definition of _relative minimum_ is analogous.
