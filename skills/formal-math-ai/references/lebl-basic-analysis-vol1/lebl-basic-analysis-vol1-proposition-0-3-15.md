**Proposition 0.3.15.** _Consider 𝑓_ : _𝐴_ → _𝐵. Let 𝐶, 𝐷 be subsets of 𝐵. Then_

Read the last line of the proposition as _𝑓_[−][1] ( _𝐵_ \ _𝐶_ ) = _𝐴_ \ _𝑓_[−][1] ( _𝐶_ ).

> _Proof._ We start with the union. If _𝑥_ ∈ _𝑓_[−][1] ( _𝐶_ ∪ _𝐷_ ), then _𝑥_ is taken to _𝐶_ or _𝐷_ , that is, _𝑓_ ( _𝑥_ ) ∈ _𝐶_ or _𝑓_ ( _𝑥_ ) ∈ _𝐷_ . Thus _𝑓_[−][1] ( _𝐶_ ∪ _𝐷_ ) ⊂ _𝑓_[−][1] ( _𝐶_ ) ∪ _𝑓_[−][1] ( _𝐷_ ). Conversely, if _𝑥_ ∈ _𝑓_[−][1] ( _𝐶_ ), then _𝑥_ ∈ _𝑓_[−][1] ( _𝐶_ ∪ _𝐷_ ). Similarly for _𝑥_ ∈ _𝑓_[−][1] ( _𝐷_ ). Hence _𝑓_[−][1] ( _𝐶_ ∪ _𝐷_ ) ⊃ _𝑓_[−][1] ( _𝐶_ ) ∪ _𝑓_[−][1] ( _𝐷_ ), and we have equality.

The rest of the proof is left as an exercise.
