Proposition 7.2.22

**Proposition 7.2.22.** _Let_ ( _𝑋, 𝑑_ ) _be a metric space and 𝐴_ ⊂ _𝑋. Then 𝑥_ ∈ _𝐴 if and only if for every 𝛿>_ 0 _, 𝐵_ ( _𝑥, 𝛿_ ) ∩ _𝐴_ ≠ ∅ _._

> _Proof._ Let us prove the two contrapositives. Let us show that _𝑥_ ∉ _𝐴_ if and only if there exists a _𝛿>_ 0 such that _𝐵_ ( _𝑥, 𝛿_ ) ∩ _𝐴_ = ∅.

_𝑐_ First suppose _𝑥_ ∉ _𝐴_ . We know _𝐴_ is closed. Thus there is a _𝛿>_ 0 such that _𝐵_ ( _𝑥, 𝛿_ ) ⊂ _𝐴_ . _𝑐_ As _𝐴_ ⊂ _𝐴_ we see that _𝐵_ ( _𝑥, 𝛿_ ) ⊂ _𝐴_ ⊂ _𝐴𝑐_ and hence _𝐵_ ( _𝑥, 𝛿_ ) ∩ _𝐴_ = ∅.

On the other hand, suppose there is a _𝛿>_ 0 such that _𝐵_ ( _𝑥, 𝛿_ ) ∩ _𝐴_ = ∅. In other words, _𝐴_ ⊂ _𝐵_ ( _𝑥, 𝛿_ ) _[𝑐]_ . As _𝐵_ ( _𝑥, 𝛿_ ) _[𝑐]_ is a closed set, as _𝑥_ ∉ _𝐵_ ( _𝑥, 𝛿_ ) _[𝑐]_ , and as _𝐴_ is the intersection of closed sets containing _𝐴_ , we have _𝑥_ ∉ _𝐴_ . □

We can also talk about the interior of a set (points we cannot approach from the complement), and the boundary of a set (points we can approach both from the set and its complement).
