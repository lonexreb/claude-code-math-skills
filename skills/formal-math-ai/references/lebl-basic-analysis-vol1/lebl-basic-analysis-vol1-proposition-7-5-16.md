Proposition 7.5.16

**Proposition 7.5.16.** _Let_ ( _𝑋, 𝑑𝑋_ ) _and_ ( _𝑌, 𝑑𝑌_ ) _be metric spaces, 𝑆_ ⊂ _𝑋, 𝑝_ ∈ _𝑋 a cluster point of 𝑆, and let 𝑓_ : _𝑆_ → _𝑌 be a function such that 𝑓_ ( _𝑥_ ) _converges as 𝑥 goes to 𝑝. Then the limit of 𝑓_ ( _𝑥_ ) _as 𝑥 goes to 𝑝 is unique._
