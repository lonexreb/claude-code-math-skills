**Theorem 1.2.1.** _There exists a unique ordered field_ ℝ _with the such that_ ℚ ⊂ ℝ _._

Note also that ℕ ⊂ ℚ. We saw that 1 _>_ 0. By (exercise), we can prove that _𝑛 >_ 0 for all _𝑛_ ∈ ℕ. Similarly, we verify simple statements about rational numbers. For example, we proved that if _𝑛 >_ 0, then[1] / _𝑛 >_ 0. Then _𝑚 < 𝑘_ implies _[𝑚]_ / _𝑛 <[𝑘]_ / _𝑛_ .
